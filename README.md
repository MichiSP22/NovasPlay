# NovasPlay

NovasPlay es una plataforma web para recargas de videojuegos, construida con Angular y enfocada en una experiencia rapida, clara y adaptable para usuarios y administradores.

## Desarrollo

Para iniciar el proyecto en local:

```bash
npm start
```

Luego abre `http://localhost:4200/`.

## Build

Para compilar el proyecto:

```bash
npm run build
```

## Tecnologias

- Angular
- TypeScript
- HTML y CSS
- SSR con Angular
