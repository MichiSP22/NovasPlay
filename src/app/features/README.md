# Features

Acciones o flujos de usuario reutilizables.

Nota: esta carpeta ya existe con un componente llamado `features`. Antes de usarla como capa arquitectonica completa, conviene migrar ese componente a una ubicacion mas especifica, por ejemplo `pages/home/features-section`.

Ejemplos futuros:

- `auth/login`
- `auth/register`
- `product/create-product`
- `product/edit-product`
